Installation Instructions on Windows PC

  1. Download the APK: Click the link above and download the APK file to your Windows PC.
  2. Enable USB Debugging on Your Android Device:
    • Open the Settings on your Android device.
    • Navigate to About phone and tap the Build number 7 times to enable Developer Options.
    • Go back to Settings, then open Developer options and enable USB debugging.
  3. Connect Your Device: Use a USB cable to connect your Android device to your Windows PC.
  4. Transfer the APK:
    • You can simply drag and drop the APK file into your device’s storage using File Explorer.
    • Alternatively, you can use tools like ADB for installation: 
      adb install path/to/your.apk
  5. Install the APK: Once the APK is transferred, locate it using a file manager on your Android device and tap to install it. You may need to enable installation from unknown sources in your device’s security settings.

If you encounter any issues during the installation process, please refer to the Android developer documentation or reach out for support.
